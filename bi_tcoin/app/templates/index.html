O que é, afinal, um bitcoin?

Bitcoin é uma criptomoeda desenvolvida por Satoshi Nakamoto, projetada para ser descentralizada e de código aberto. Mas o quê tudo isso significa?
Significa que esta moeda online não sofre influência direta de intervenções governamentais e qualquer um interessado pode auditar as transações. Isto torna a moeda mais independente e traz uma série de benefícios!
Como o nosso foco aqui é botar a mão na massa, quem quiser saber mais detalhes a respeito, pode acessar os links a seguir:

  • Bircoin Brasil
  • Video explicativo

Como você pode intergir com esse mercado

A imagem profissional abaixo vai ensiná-lo quais os principais postos de trabalho dentro da rede de bitcoins atualmente e quais as contribuições que podem ser feitas.

  • 1)Algoritmo principal Bitcoin:
    • Desenvolvimento de código-fonte: O código de desenvolvimento do bitcoin é aberto e todos podem contribuir para a melhoria das funcionalidades. É só acessar e contribuir: GitHub
      PS: se você não sabe nada de GitHub, sugiro que acesse: link
  • 2)Blockchain: O Blockchain é o protocolo pelo qual a rede trabalha. Se você leu os links anteiores, sabe que sua possibilidade de auditoria por qualquer um o torna uma ferramenta incrível. Para contribuir, basta fazer da mesma forma que para o código-fonte.
    Para saber mais: What is Blockchain?
  • 3)Testnet3: A rede testnet3 é utilizada para testes de novos códigos. Se você criou uma nova aplicação e precisa saber se funciona, UTILIZE ESSA REDE! Isso evita que você perca seus bitcoins por algum erro de código.
    Wiki do Bitcoin: Link
  • 4)Mineração de Bitcoins: Se você quer descobrir os próximos bitcoins no mundo, seja um minerador! De uma maneira beeeeem simples, minerar Bitcoins é "emprestar" processamento de suas máquinas para que a rede funcione. Em troca desta ajuda, o sistema te remunera com alguns décimos de bitcoins. Infelizmente grandes corporações criaram clusters gigantescos com diversas máquinas e um processamento poderoso, inviabilizando os marinheiros de primeira viagem a colocar seu próprio hardware e começar a minerar. Felizmente, existem algumas outras opções como financiar um processamento através da internet (mineração em nuvem) e também os pools, onde diversos mineradores independentes se unem para minerar em conjunto. O link a seguir é um excelente guia: Hobby Mining
  • 5)Cloud Mining: Para quem quer apenas investir sem muitas preocupações de hardware, programação, gastos com energia e outros, é possível alugar hardware em nuvem para que outra empresa cuide destes problema pra você. Basicamente é possível pagar um valor mensal, único ou anual e depois só colher os resultados. Um exemplo deste tipo de empresa é a Genesis Mining
  • 6)Pool: Os pools são grupos de mineradores independentes que se uniram para fazer frente às grandes mineradoras corporativas dos dias atuais. A Slush Pool, primeiro pool de mineradores de Bitcoin do mundo pode ser acessada neste link: Link
  • 7) Mineração independente: Para quem insiste em ser um lobo solitário, ainda existe salvação. É possível comprar um computador ASICs especializado para a mineração de bitcoins e deixar ele rodando. Com um pouco mais de conhecimento de eletrônica (e coragem!) é possível também construir um. Vale lembrar que é preciso fazer um bom cálculo de retorno de investimento, como os gastos de energia elétrica, por exemplo. Quem quiser entender mais sobre essa arquitetura, pode acessar: Bitcoin wiki - ASICs
  • 8) Bolsa de valores: Se existem especulações sobre o Bitcoin e também variações desta moeda em relação às outras, existe a possibilidade de ganhar dinheiro com esta flutuação. Pensando nisso, diversas empresas foram criadas para funcionar como uma espécie de bolsa de valores, através de plataformas de compra e venda de bitcoins. Um exemplo destas empresas é a brasileira FoxBit
  • 9) Troca direta: Como o bitcoin pode ser trocado diretamente entre os usuários, é possível também que você faça e receba pagamentos sem intermediários, reduzindo custos e também dando mais liberdade de atuação. Para este tipo de ação (na verdade para qualquer transação de bitcoin) é preciso ter uma carteira de bitcoins. Depois disso, é só você passar um endereço para o outro usuário e através deste endereço ele consegue transferir os bitcoins pra você! É simples e seguro. Um bom site com carterias para Bitcoins é a Blockchain Wallet
  • 10) Prestação de serviços: Aqui o céu é o limite! Existem diversos tipos de serviços/ produtos que podem ser trocados por Bitcoins. Empresas que te pagam em Bitcoins para assistir propagandas, pagamento em Bitcoins para cartões de crédito, compra de jogos e muito mais! É bem simples de se aplicar. Para saber mais sobre como construir um sistema usando APIs para a rede Bitcoin, acesse nossos tutoriais de programador.
  • 11) Queima de Bitcoins: Apesar de ser uma espécie de prestação de serviços também, preferi separar os assuntos para que fosse possível explicar este tópico em detalhes. Os Block Chains são completamente rastreáveis e seus registros são vitalícios, ou seja, se você marcar alguma informação lá, ela vai estar lá para sempre. Muitas pessoas estão utilizando esta propriedade para registrarem informações de marca, contratos e diversas outras informações que podem ser utilizadas futuramente. Isto abre espaço para um novo campo de serviços. Vale lembrar apenas que você pode queimar poucos satoshis (menor unidade de bitcoins existentes) por requisição. Chamamos isso de queimar Bitcoins pois os Bitcoins gastos desta maneira não podem jamais ser recuperados. Saiba mais: Link
